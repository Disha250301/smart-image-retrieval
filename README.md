# smart-image-retrieval
MCA project: deep learning and hypergraph-based image retrieval system with Streamlit UI

## Project Structure

- `indexing.py` – builds the image index / feature database  
- `ranking.py` – ranks images based on similarity  
- `evaluate.py` – evaluation scripts (Precision, Recall, mAP)  
- `evaluate_app.py` – evaluation with the Streamlit app  
- `generate_groundtruth.py` – creates ground truth labels for evaluation  
- `evaluation_results.csv` – stored evaluation metrics  
- `monuments.csv` – sample metadata / dataset information  
- `practice.py`, `demo.py`, `test.py` – experimental and demo scripts  
- `temp_query.jpg` – example query image
